# 로그인/회원가입 페이지 구현

## 작업 내용
- /login 페이지: 이메일/비밀번호 로그인 + 카카오/네이버 소셜 로그인
- /register 페이지: 2단계 회원가입 (약관 동의 → 정보 입력)
- 모바일 앱 LoginActivity, JoinUserInfoActivity, TermsActivity 기반

## 주요 기능
- 로그인: 이메일/비밀번호 입력, 비밀번호 표시/숨김, 비밀번호 찾기 링크
- 소셜 로그인: 카카오(노란색), 네이버(초록색) 브랜딩 버튼
- 회원가입 Step 1: 전체 동의 + 개별 약관 (필수 3개 + 선택 1개)
- 회원가입 Step 2: 이메일, 닉네임, 비밀번호(8자+), 비밀번호 확인
- 스텝 인디케이터 UI (1 약관 동의 → 2 정보 입력)
