# 홈 화면 모바일 앱 레이아웃 재구성

## 작업 내용
AOS 앱 스크린샷 기준으로 홈 화면 레이아웃을 모바일 앱과 동일한 구조로 재배치.
웹에서는 narrow 컨테이너(800px)로 내로우하게 배치하되, 반응형은 유지.

## 변경 사항
- HeroSection 제거 → 검색바를 콘텐츠 영역에 배치 (스크롤 시 헤더로 이동)
- 업태 카테고리: 가로 스크롤 → 4열 2줄 그리드
- 랜딩 버튼: 글래스모피즘 카드 → 3열 그리드
- 배너: 비율 2.5:1 적용
- 최근 본 숙소/매거진: 마우스 드래그 지원, overflow-padding 분리
- 추천 숙소: 밑줄 탭 + 쿠폰배지/하트/할인율 카드
- 기획전: 가로 슬라이더 → 세로 스택
- GNB: 4탭 → 5탭 (홈/내주변/찜숙소/혜택함/마이페이지)
- section-px 유틸리티로 전 섹션 좌우 여백 통일
- 배경 흰색 통일
