# 검색 모달 단일 화면으로 재구성 (모바일 앱 캡쳐 기반)

- **날짜**: 2026-03-20
- **작업 유형**: refactor
- **영향 범위**: src/components/search/SearchModal.tsx

## 변경 사항
- 3-step 플로우 (여행지→날짜→인원) → **단일 화면**으로 전면 재구성
- 모바일 앱 검색 화면과 동일한 레이아웃:
  1. ← 뒤로가기 + "검색" 타이틀
  2. 텍스트 입력 필드 (🔍 지역, 전철역, 숙소명으로 찾아보세요)
     - 모달 오픈 시 자동 포커스
     - Enter 키로 검색 실행
     - X 버튼으로 입력 초기화
  3. 날짜/인원 조건 바 (3.20(금) - 3.21(토) · 1박 | 2명)
     - 인원 클릭 시 Popover로 성인/아동 카운터
  4. 최근 검색 (최근 검색어 목록, 개별 삭제, 전체 삭제)
     - 비어있으면 "최근 검색한 조건이 없습니다" 표시
  5. 인기 검색어 순위 (1~8위, 오늘 날짜 기준)
     - 1~3위 primary 색상 강조
     - sm 이상에서 2열 그리드

## 기술적 결정
- step 분리 제거: 모바일 앱에서도 한 화면에 검색/최근/인기가 모두 보이므로 동일하게 적용
- 검색어 입력 시 Enter → /search?keyword= 로 이동
- 인기 검색어 클릭 시에도 동일한 플로우
- 지역 선택 그리드는 제거 (검색어 입력 또는 /search 페이지의 SearchConditionBar에서 선택)
