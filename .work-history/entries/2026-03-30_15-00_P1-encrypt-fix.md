# AES-128-CBC 암호화 구현 — app-secret-code + enc_password

## 문제
- 로그인 API 호출 시 401 토큰 불일치 에러
- `app-secret-code` 헤더에 raw secret을 보내고 있었으나, 실제로는 AES 암호화 값 필요

## 원인 분석
- AOS 앱 `EncryptUtil.kt` + `NetworkHeaderUtil.kt` 분석
- `app-secret-code` = AES-CBC(accessToken + "-" + YYYYMMddHHmmss, secret)
- `enc_password` = AES-CBC(password, secret)

## 해결
- `encrypt.ts` 신규 생성: AES-128-CBC + PKCS7 + Base64 (Web Crypto API)
- `client.ts`: 매 요청마다 generateSecretCode()로 헤더 생성
- LoginPage/RegisterPage: encryptPassword()로 비밀번호 암호화
