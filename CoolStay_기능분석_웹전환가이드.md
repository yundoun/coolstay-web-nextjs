## 1. 서비스 개요

CoolStay는 **모텔/숙박 예약 플랫폼**으로, 사용자가 지역/키워드/현재위치 기반으로 숙소를 검색하고, 대실(시간제) 또는 숙박(1박 이상) 형태로 객실을 예약·결제할 수 있는 서비스이다.

### 핵심 도메인

| 도메인 | 설명 |
|--------|------|
| 숙소 검색 | 지역별, 역 주변, 키워드, 현재위치 기반 숙소 탐색 |
| 숙소/객실 정보 | 이미지, 편의시설, 리뷰, 이벤트, 요금, 위치 |
| 예약/결제 | 대실·숙박 예약, 쿠폰·마일리지 할인, PG 결제·현장결제 |
| 회원 관리 | 이메일/소셜 로그인, 프로필 관리, 회원탈퇴 |
| 혜택 | 쿠폰, 마일리지 적립/사용, 이벤트 |
| 고객 지원 | 리뷰, 1:1 문의, FAQ, 공지사항, 가이드 |

### 사용자 유형별 기능 차이

| 기능 | 비로그인 | 로그인 회원 |
|------|----------|------------|
| 숙소 검색/상세 보기 | O | O |
| 예약 | O (비회원 예약) | O |
| 찜하기 | X | O |
| 쿠폰 (마이쿠폰) | X | O |
| 마일리지 | X | O |
| 리뷰 작성 | X | O |
| 1:1 문의 | X | O |
| 예약 내역 조회 | 예약번호+전화번호 조회 | 목록 조회 |
| 알림센터 | X | O |
| 최근 본 숙소 | O | O |
| 찜 목록 | X | O |

## 2. 온보딩 & 인증

### 2.1 스플래시 화면

- **표시 내용**: 앱 로고 + 브랜딩 이미지
- **자동 처리**:
  - 앱 강제 업데이트 필요 시: 업데이트 안내 팝업 → Play Store 이동
  - 최초 실행 시: 권한 동의 팝업 → OS 위치 권한 요청 → 알림 설정 안내
  - 자동 로그인 설정 시: 저장된 토큰으로 자동 로그인 시도
  - 전화번호 미등록 시: 전화번호 인증 화면으로 이동
  - 앱 잠금 설정 시: 앱 비밀번호 입력 화면으로 이동
- **웹 전환**: 스플래시 불필요, 자동 로그인은 쿠키/세션으로 대체

### 2.2 튜토리얼

- **표시 내용**: 3페이지 스와이프 가이드 이미지
- **사용자 액션**: 좌우 스와이프, "다음" 버튼, 마지막 페이지에서 "시작" 버튼
- **동작**: 최초 1회만 표시, 완료 후 로그인 화면으로 이동
- **웹 전환**: 랜딩 페이지 또는 온보딩 모달로 대체 가능

### 2.3 로그인

- **표시 내용**:
  - 이메일 입력 필드
  - 비밀번호 입력 필드 (마스킹)
  - "로그인" 버튼 (이메일·비밀번호 모두 입력 시 활성화)
  - "비밀번호 찾기" 링크
  - "카카오 로그인" 버튼 (노란색)
  - "네이버 로그인" 버튼 (초록색)
  - "이메일 회원가입" 링크
  - 닫기(X) 버튼

- **사용자 액션**:
  - 이메일 + 비밀번호 로그인
  - 카카오 OAuth 로그인
  - 네이버 OAuth 로그인
  - 비밀번호 찾기 이동
  - 회원가입 이동
  - 화면 닫기

- **분기 처리**:
  - 로그인 성공 → 메인 화면
  - 전화번호 미등록(상태 S2) → 전화번호 인증 화면
  - SNS 로그인 시 미가입 계정 → 자동으로 회원가입 플로우로 이동

### 2.4 회원가입 플로우

#### Step 1: 약관 동의

- **표시 내용**:
  - "전체 동의" 체크박스
  - 필수/선택 표시가 있는 약관 목록 (서버에서 제공)
  - 각 약관 제목 탭 → WebView로 전문 보기
- **사용자 액션**: 개별/전체 체크, 약관 전문 보기
- **진행 조건**: 모든 필수 약관 동의 시 다음 버튼 활성화

#### Step 2: 휴대폰 인증

- **표시 내용**:
  - 휴대폰 번호 입력 필드
  - 인증번호 입력 필드 (코드 발송 후 표시)
  - 카운트다운 타이머
  - 재발송 링크
- **사용자 액션**: 번호 입력 → 인증코드 발송 → 코드 입력 → 확인
- **제한 사항**: 발송 최대 5회, 재발송 간 1분 쿨다운
- **초과 시**: "최대 횟수 초과" 팝업 → 화면 종료

#### Step 3: 정보 입력

- **이메일 가입 시**: 이메일, 닉네임, 비밀번호, 비밀번호 확인 (4개 필드)
- **SNS 가입 시**: 닉네임만 입력 (1개 필드)
- **유효성 검사**: 이메일 형식, 중복 이메일, 비밀번호 규칙, 비밀번호 일치
- **이탈 방지**: 입력 내용이 있을 때 뒤로가기 시 확인 팝업

### 2.5 비밀번호 찾기

#### Step 1: 이메일 입력

- 등록된 이메일 주소 입력 → 서버에서 인증수단 조회

#### Step 2: 인증수단 선택

- 이메일 인증 (마스킹된 이메일 표시) / SMS 인증 (마스킹된 전화번호 표시) 중 택 1
- 이메일 선택 시: 비밀번호 재설정 링크 발송 → 확인 팝업 → 로그인 화면
- SMS 선택 시: 인증코드 입력 화면으로 이동

#### Step 3: 인증코드 입력 (SMS 선택 시)

- 인증코드 입력 + 카운트다운 타이머
- 재발송 (5회 제한, 1분 쿨다운)
- 인증 성공 → "임시 비밀번호가 발송되었습니다" 팝업 → 로그인 화면

## 3. 홈 화면

### 3.1 상단 헤더 (접이식)

| 요소 | 설명 | 탭 시 동작 |
|------|------|-----------|
| 앱 로고 | 좌측 상단 | - |
| 지역명 | 현재 검색 지역 (예: "대구역") + 드롭다운 화살표 | 검색 탭으로 이동 |
| 날짜 범위 | 체크인~체크아웃 (예: "4.12(월) ~ 4.13(화)") + 캘린더 아이콘 | 캘린더 피커 열기 |
| 인원 수 | 성인/아동 수 (예: "성인2 아동1") + 인원 아이콘 | 인원 선택 바텀 팝업 |
| 정렬 아이콘 | 스크롤 시 애니메이션으로 등장 | 정렬 옵션 팝업 |
| 지도 버튼 | 우측 상단 플로팅 | 현재위치 지도 검색 화면 이동 |

### 3.2 콘텐츠 영역

| 영역 | 설명 |
|------|------|
| 이벤트 배너 캐러셀 | 가로 자동 슬라이드 프로모션 이미지, 탭 시 이벤트 상세/랜딩 이동 |
| 카테고리 필터 | 이벤트 기반 필터 칩, 탭 시 숙소 목록 필터링 |
| 마일리지 안내 카드 | 닫기 가능한 안내 버블 (1회 표시 후 다시 안 보임) |
| 숙소 목록 | 카드형 리스트, 12개씩 무한스크롤, 당겨서 새로고침 |
| 빈 상태 | 검색 결과 없을 때 안내 메시지 |
| 푸터 | 사업자 정보 |

### 3.3 정렬 옵션

| 정렬 | 설명 | 비고 |
|------|------|------|
| 거리순 | 현재 위치에서 가까운 순 | 위치 기반 검색일 때만 활성화 |
| 평점 높은순 | 리뷰 평점 높은 순 | |
| 가격 높은순 | 객실 가격 높은 순 | |
| 가격 낮은순 | 객실 가격 낮은 순 | |
| 마일리지 적립률 높은순 | 마일리지 적립률 기준 | |
| 관심 등록 많은순 | 찜 수 기준 | |

### 3.4 진입 시 특수 동작

- **위치 미설정 시**: LocationSelectionActivity가 강제 표시 (지역 선택 필수)
- **프로모션 팝업**: 서버에서 내려주는 공지 팝업 표시
- **리뷰 요청 팝업**: 과거 이용건에 대한 리뷰 작성 유도

## 4. 검색

### 4.1 지역별 검색

- **구조**: 대분류 지역 (가로 스크롤 인디케이터) → 소분류 지역 (세로 리스트)
- **대분류 예시**: 서울, 부산, 대구, 인천, 광주, 대전, 울산, 경기, 강원...
- **동작**: 대분류 탭 → 소분류 로드 → 소분류 탭 → 홈으로 이동 (검색 결과 갱신)

### 4.2 역 주변 검색

- **구조**: 도시 (가로 스크롤) → 노선 목록 (좌측 컬럼) → 역 목록 (우측 컬럼)
- **동작**: 도시 선택 → 노선 선택 → 역 선택 → 홈으로 이동 (검색 결과 갱신)
- **예외**: 지하철 데이터 없는 도시는 빈 상태 표시

### 4.3 키워드 검색

#### 기본 상태: 인기 키워드

- 날짜 선택 버튼 + 인원 선택 버튼
- 해시태그 형태 편의시설 키워드 그리드 (예: #수영장, #바베큐, #스파 등)
- 키워드 탭 시 해당 편의시설로 숙소 검색

#### 키보드 활성 시: 최근 검색어

- 검색 이력 목록 (키워드명 + 날짜범위 + 인원수)
- 개별 삭제 (X 버튼) / 전체 삭제
- 과거 날짜의 검색어는 오늘 날짜로 자동 보정
- 숙소 타입(M) 탭 시 → 숙소 상세로 바로 이동, 그 외 → 검색 결과 화면

#### 검색어 입력 후: 추천 키워드

- 검색어와 일치하는 결과 목록 (매칭 글자 하이라이트)
- 결과 유형: 지역, 지하철, 제휴점(숙소)
- 지역/지하철 탭 → 검색 결과 화면, 숙소 탭 → 숙소 상세로 이동

### 4.4 검색 결과 목록

- **상단**: 검색 타이틀 (지역명/키워드) + 재검색 아이콘
- **필터바**: 날짜 선택 + 인원 선택 + 정렬 아이콘
- **숙소 리스트**: 카드형, 12개씩 무한스크롤, 당겨서 새로고침
- **정렬**: 6가지 옵션 (위치 기반 검색 시 기본값 거리순, 그 외 랜덤)
- **빈 상태**: 검색 결과 없음 안내

### 4.5 현재 위치 검색 (지도)

- **표시**: 네이버 지도 전체 화면 + 중앙 핀 + 현재위치 점
- **주소 표시**: 지도 중심 이동 시 역지오코딩으로 주소 자동 갱신
- **기본 좌표**: GPS 미사용 시 대구역 (35.8761, 128.5938)
- **사용자 액션**:
  - 지도 이동/확대 → 중앙 핀 위치의 주소 자동 표시
  - "내 위치" 버튼 → GPS 위치로 이동 (위치 권한 필요)
  - "적용" 버튼 → 해당 좌표를 검색 기준으로 설정 후 홈 이동

## 5. 숙소 상세

### 5.1 화면 구성 (위→아래)

| 영역 | 내용 |
|------|------|
| **이미지 슬라이드쇼** | 숙소 이미지 갤러리, 탭 시 전체화면 뷰어 |
| **기본 정보** | 숙소명, 위치 설명, 마일리지 적립률(예: "+3%"), 보유 마일리지(로그인 시) |
| **평점/리뷰** | 별점 + 리뷰 수 (최소 기준 이상일 때만 표시) |
| **이벤트 배너** | 진행중 이벤트 제목 + 추가 이벤트 수 (예: "+2") |
| **쿠폰 영역** | "즉시할인 쿠폰" 받기 버튼 + "내 쿠폰" 보기 버튼 (로그인 시) |
| **베스트 리뷰** | 대표 리뷰 1건 (작성자명, 별점, 내용) |
| **객실 목록** | 각 객실의 이미지, 이름, 가격, 가용 상태 (대실/숙박 구분) |
| **편의시설** | 가로 스크롤 아이콘 목록 |
| **사장님 인사말** | 업주 메시지 |
| **리뷰 목록** | 리뷰 리스트 + "더보기" 버튼 |
| **주차 정보** | 주차 가능 여부, 주차장 수, 상세 정보 |
| **혜택/이용안내** | 객실 혜택, 추가 혜택 |
| **지도** | 네이버 지도 프리뷰 + 주소 + 주소복사 |

### 5.2 사용자 액션

| 액션 | 설명 | 조건 |
|------|------|------|
| 이미지 전체화면 | 이미지 탭 시 갤러리 뷰어 | - |
| 찜하기 | 하트 아이콘 토글 | 로그인 필요 (비로그인 시 로그인 팝업) |
| 공유 | 카카오톡 링크 공유 | - |
| 이벤트 보기 | 이벤트 배너 탭 → 이벤트 목록 | - |
| 쿠폰 받기 | 즉시할인 쿠폰 다운로드 | - |
| 내 쿠폰 | 보유 쿠폰 목록 | 로그인 필요 |
| 객실 선택 | 객실 카드 탭 → 객실 상세 | - |

### 5.3 숙소 위치 상세 (MotelLocationActivity)

## 7. 예약 플로우

### 7.1 Step 1: 예약 정보 입력

#### 이용 시간

| 유형 | 표시 |
|------|------|
| 대실 | 가로 스크롤 시간대 선택 (기본: 가장 빠른 시간) |
| 숙박 | 체크인/체크아웃 날짜 + 시간 표시 |

- 대실: 당일 대실 종료시간이 이미 지난 경우 경고 팝업 + 강제 이탈
- 대실: 선택 시간이 기본 대실 시간보다 짧을 경우 경고 표시

#### 예약자 정보

- 이름 + 휴대폰번호 표시
- 안심번호 적용 안내
- 탭 시 예약자 정보 수정 화면 이동
- **비로그인 시**: 로그인 팝업 (비회원 예약 진행 옵션 포함)

#### 예약자 정보 수정 (EditBookerInfoActivity)

1. 이름 입력 (유효성 검사)
2. 전화번호 입력 → "인증번호 요청" 버튼
3. SMS 인증코드 발송 → 카운트다운 타이머 → 코드 입력 → 확인
4. 인증 완료 후 전화번호 잠금
5. "전화번호 변경" 버튼으로 재인증 가능
6. "예약 계속하기" 버튼 (이름 유효 + 전화번호 인증 완료 시 활성화)

#### 교통수단

- 도보 / 자가용 라디오 선택
- 주차장 없음 → "자가용" 비활성, "주차장 없음" 메시지
- 주차장 만차 → "현재 만차, 주차가 어려울 수 있습니다" 안내
- 당일 자가용 예약 → 주차 안내 가이드

#### 진행 조건

- 예약자 정보 입력 완료 + 교통수단 선택 시 "다음" 버튼 활성화
- 인원 초과 시 토스트 경고

### 7.2 Step 2: 결제

#### 쿠폰

| 쿠폰 유형 | 설명 | 조건 |
|-----------|------|------|
| 선착순 쿠폰 | 숙소에서 제공하는 즉시 할인 쿠폰 | - |
| 마이 쿠폰 | 개인 보유 쿠폰 | 로그인 필요 |

- 각각 탭 → 쿠폰 선택 화면 이동

#### 마일리지 (로그인 시)

- 현재 마일리지 잔액 표시
- "최대 사용" 체크박스
- 직접 입력 필드 (숫자)
- **제한**: 최소 사용 금액 3,000원, 최소 단위 1,000원
- **비로그인 시**: "로그인하고 마일리지 사용하기" 버튼

#### 가격 상세

| 항목 | 설명 |
|------|------|
| 판매가 | 객실 기본 가격 |
| 특별할인 | 숙소 자체 할인 (있을 경우) |
| 쿠폰 1 할인 | 선착순 쿠폰 적용 금액 |
| 쿠폰 2 할인 | 마이 쿠폰 적용 금액 |
| 마일리지 할인 | 마일리지 사용 금액 |
| **최종 결제금액** | 모든 할인 적용 후 금액 |

#### 결제 방식

| 방식 | 설명 | 조건 |
|------|------|------|
| 바로결제 | PG 결제 (신용카드/계좌이체) | 기본 |
| 현장결제 | 숙소 방문 시 결제 | 업소 허용 시만 표시 |

- **노쇼 이력 사용자**: 현장결제 차단 + 경고 메시지

#### 결제 수단 선택 (SelectPaymethodActivity)

| 수단 | PG사 |
|------|------|
| 신용/체크카드 | INICIS |
| 실시간 계좌이체 | INICIS |

#### 약관 동의 (모두 필수)

1. 전체 동의 마스터 체크박스
2. 숙소 이용규칙 및 취소/환불 규정 (상세 보기 링크)
3. 개인정보 수집·이용 동의 (상세 보기 링크)
4. 개인정보 제3자 제공 동의 (상세 보기 링크)
5. 만 19세 이상 확인 (비로그인 시만, 상세 보기 링크)

#### 결제 진행

- "결제하기 {금액}" 또는 "현장결제 {금액}" 버튼
- **활성 조건**: 결제수단 선택 + 모든 약관 동의
- 탭 시 최종 예약 확인 팝업 → 확인 시 진행
- **뒤로가기**: 변경사항이 있으면 "변경 내용을 버릴까요?" 확인 팝업

### 7.3 결제 처리

| 경우 | 처리 |
|------|------|
| 결제 금액 > 0 + 바로결제 | iamport + INICIS PG WebView 결제 |
| 결제 금액 = 0 | PG 없이 바로 예약 완료 |
| 현장결제 | PG 없이 바로 예약 완료 |

### 7.4 예약 완료

- **표시 내용**:
  - 예약 완료 아이콘 + 메시지
  - "예약내역에서 확인하세요" 안내
  - 마일리지 적립 안내
  - "48시간 후 리뷰 작성하면 마일리지 적립" 안내
  - **현장결제 시**: 노쇼 패널티 경고 문구

- **버튼**: "홈으로" / "예약 상세 보기"

## 8. 마이페이지 (더보기)

### 8.1 메인 화면

#### 로그인 사용자

- 닉네임 표시
- 쿠폰 수 (탭 → 쿠폰 화면)

- 알림 벨 아이콘 + 읽지 않은 알림 뱃지 (탭 → 알림센터)
- 고객센터 전화번호 (탭 → 전화 걸기)

#### 비로그인 사용자

- "로그인/가입" 버튼 (탭 → 로그인 화면)
- 축소된 메뉴 (비회원 예약확인, 최근본 상품, 약관, 공지, 이벤트, 가이드, FAQ, 설정)

### 8.2 메뉴 상세

#### 예약/이용 내역 (로그인)

- **탭 구성**: "전체" (전체 이력, 무한스크롤) / "예약중" (진행중인 예약만)
- **숙소 카드 정보**: 숙소명, 객실명, 이용유형, 날짜, 상태
- **액션**: 카드 탭 → 예약 상세, "더보기" → 공유/삭제, 예약 취소
- **전체삭제**: 타이틀바 삭제 버튼 → 확인 팝업 → 전체 삭제
- **예약 취소**: 확인 팝업 (현장결제 시 패널티 경고)
- **빈 상태**: 안내 메시지 + "홈으로" 버튼

#### 비회원 예약 확인 (비로그인)

- 예약번호 입력 (숫자, 최대 12자리) + 전화번호 입력
- 조회 → 예약 상세 화면
- "로그인" 링크 → 로그인 후 예약내역 화면으로 리다이렉트

#### 최근본/찜 상품

- **로그인**: "최근본" / "찜" 2개 탭 + 삭제 모드
- **비로그인**: "최근본" 1개 탭만
- **삭제 모드**: 체크박스 표시 → 선택 삭제 / 전체 삭제
- **카드 탭**: 숙소 상세로 이동
- 최근본: 로컬 저장소에서 관리
- 찜: 서버 API 연동

#### 쿠폰

- 쿠폰 코드 입력 + "등록" 버튼 (옵션 선택 팝업 있을 수 있음)
- 총 쿠폰 수 표시
- 쿠폰 카드 목록 (제목, 할인금액, 유효기간, 사용 조건)
- 카드 탭 → 쿠폰 상세
- 빈 상태: "쿠폰이 없습니다" + "이벤트 보러가기" 버튼
- 쿠폰 정책 WebView 버튼

#### 쿠폰 상세

- 쿠폰 제목, 할인금액, 유효기간, 사용 조건 설명, 주의사항
- 특정 숙소 지정 쿠폰: "지금 예약하기" 버튼 → 해당 숙소 상세

#### 마일리지

- 매장별 마일리지 목록: 숙소 이미지, 숙소명, 위치, 적립률, 누적금액, 소멸예정금액
- 카드 탭 → 마일리지 상세
- 숙소 이미지 탭 → 숙소 상세
- 빈 상태: "마일리지가 없습니다" + "홈으로" 버튼
- 마일리지 정책 WebView 버튼

#### 마일리지 상세

- 현재 마일리지, 총 누적, 소멸예정 금액
- 마일리지 이력 목록: 객실유형, 날짜, 상태, 금액, 잔액
- "지금 예약하기" 버튼 → 숙소 상세

#### 리뷰 관리

- 평균 점수, 총 리뷰 수, 작성 가능 리뷰 수
- 리뷰 카드 목록 (숙소 정보, 별점, 내용, 이미지)
- "더보기" → 수정 / 삭제
- 빈 상태: "리뷰가 없습니다" + "예약내역 보러가기" 버튼
- 하단 "리뷰 작성" 버튼 → 예약 내역 화면 이동

#### 리뷰 상세

- 전체 리뷰 내용, 이미지 (탭 → 전체화면 뷰어)
- 숙소 정보 (탭 → 숙소 상세)
- "더보기" → 수정 / 삭제

#### 리뷰 작성/수정

- 숙소명 + 객실 유형 표시
- 별점 바 (1~5점)
- 리뷰 텍스트 입력 (최소 10자)
- 이미지 첨부 (카메라/갤러리)
- "등록" 버튼 (별점 설정 + 10자 이상 시 활성화)
- 수정 모드: 기존 리뷰 데이터 프리필

#### 1:1 문의

**작성 탭:**
- 카테고리 선택 드롭다운 (슬라이딩 팝업)
- 문의 내용 텍스트 입력 (최소 10자)
- 이미지 첨부
- "보내기" 버튼 → 성공 토스트 → 목록 탭 전환

**목록 탭:**
- 답변 상태 필터 (전체 / 대기중 / 답변완료)
- 문의 카드 목록 (카테고리 뱃지, 내용 요약)
- 카드 탭 → 문의 상세

**문의 상세:**
- 카테고리 뱃지
- 문의 내용 + 첨부 이미지 (탭 → 전체화면 뷰어)
- 답변 내용 (또는 "답변 대기중")
- 답변 시간
- "삭제" 버튼 → 확인 팝업 → 삭제

#### 이벤트

- 이벤트 카드 목록 (무한스크롤)
- **이벤트 상세**: 이벤트 이미지 + 설명 + "쿠폰 받기" 버튼
- 쿠폰 받기: 로그인 필요, 성공/이미 받음/기간 만료 팝업

#### 공지사항

- 공지 목록 (무한스크롤, 신규 뱃지)
- **공지 상세**: 제목, 날짜, 이미지, 내용, 액션 버튼 (랜딩 링크)

#### FAQ

- FAQ 목록 (무한스크롤)
- **FAQ 상세**: 제목, 날짜, 이미지, 내용

#### 이용 가이드

- 가이드 목록 (무한스크롤)
- **가이드 상세**: 가이드 내용

#### 알림센터 (로그인)

- 알림 메시지 목록, 당겨서 새로고침, 무한스크롤
- 알림 탭 → 랜딩 링크로 해당 화면 이동
- "전체 삭제" 버튼 → 확인 팝업 → 전체 삭제
- 빈 상태: "알림이 없습니다" + "설정으로 가기" 버튼

#### 모바일 설정

| 설정 | 설명 |
|------|------|
| 앱 잠금 | ON/OFF 토글 → 비밀번호 설정/해제 |
| 이용정보 푸시 알림 | ON/OFF 토글 (끄기 시 확인 팝업) |
| 이용정보 SMS 알림 | ON/OFF 토글 |
| 마케팅 푸시 알림 | ON/OFF 토글 (이용정보 푸시가 꺼지면 비활성화) |
| 앱 권한 설정 | Android 앱 상세 설정 화면 이동 |
| 앱 버전 | 현재 버전 + 업데이트 버튼 (신버전 있을 시) |

| 항목 | 이메일 로그인 | SNS 로그인 | 편집 가능 |
|------|-------------|-----------|----------|
| 로그인 유형 | - | 카카오/네이버 표시 | X |
| 이메일 | 표시 | - | X (읽기전용) |
| 닉네임 | 표시 | 표시 | O |
| 이름 | 표시 | 표시 | O |
| 전화번호 | 표시 (하이픈 포맷) | 표시 | O (SMS 재인증 필요) |
| 비밀번호 | "변경" 버튼 | - | O (현재PW + 새PW + 확인) |

### 9.3 전화번호 변경

1. 새 번호 입력
2. 인증코드 발송 (5회 제한, 1분 쿨다운)
3. 코드 입력 → 확인
4. 서버에 번호 업데이트

### 9.4 로그아웃

- 확인 팝업 → 로그아웃 → 홈 화면

### 9.5 회원 탈퇴

#### 손실 안내

- 보유 쿠폰 수, 마일리지 매장 수, 찜한 숙소 수 표시

#### 탈퇴 사유 (7개 중 택 1)

| 사유 | 대안 제안 팝업 |
|------|--------------|
| 사용이 어려워요 | FAQ + 이용 가이드 링크 |
| 서비스 불만 | 고객센터 전화 + 1:1 문의 링크 |
| 혜택이 부족해요 | 이용 가이드 + 이벤트 링크 |
| 중복 계정 | 팝업 없음 |
| 이용 기록 삭제 원해요 | 예약내역 + 검색 링크 |
| 할인 이벤트가 부족해요 | 쿠폰 + 이벤트 링크 |
| 기타 (직접입력) | 텍스트 입력 필드 활성화 |

#### 진행

- 사유 선택 + "모든 데이터가 삭제됨을 이해합니다" 동의 체크
- "탈퇴하기" 버튼 → 최종 확인 팝업 → 탈퇴 처리 → 로그아웃 → 홈 화면

## 11. 앱 전용 기능 vs 웹 대안

| 기능 | Android 구현 | 웹 대안 |
|------|-------------|---------|
| **푸시 알림** | FCM (Firebase Cloud Messaging) | Web Push API / 이메일 알림 / 인앱 알림 |
| **앱 잠금** | 4자리 비밀번호 | 불필요 (브라우저 보안에 위임) |
| **카카오톡 공유** | Kakao Android SDK | Kakao JavaScript SDK / Web Share API |
| **전화 걸기** | `tel:` Intent | `tel:` 링크 (모바일 웹 동일 동작) |
| **GPS 위치** | Android Location API | Browser Geolocation API |
| **지도** | Naver Maps Android SDK | Naver Maps JavaScript API |
| **PG 결제** | iamport Android WebView | iamport JavaScript SDK 직접 연동 |
| **앱 업데이트** | Play Store 연동 | 불필요 (항상 최신 버전) |
| **OS 권한 요청** | Android Permission Dialog | Browser Permission API (위치 등) |
| **이미지 첨부** | 카메라/갤러리 Intent | `<input type="file">` (카메라/파일 선택) |
| **자동 로그인** | SharedPreferences 토큰 저장 | 쿠키 / localStorage / 세션 |
| **딥링크/랜딩** | Intent URL Scheme | URL 라우팅 |

## 12. 전체 화면 플로우

```
[앱 진입]
  │
  ├─ 최초 실행 ──→ 튜토리얼(3P) ──→ 로그인
  │
  ├─ 자동 로그인 ON ──→ 로그인 시도 ──→ 성공 → [메인]
  │                                     └─ 실패 → [메인 (비로그인)]
  │
  └─ 자동 로그인 OFF ──→ [메인 (비로그인)]

[메인] ─── 하단 GNB 5탭 ───
  │
  ├─ [홈] ──→ 숙소 목록
  │    ├─ 지역명 탭 → [검색]
  │    ├─ 날짜 탭 → 캘린더 피커
  │    ├─ 인원 탭 → 인원 선택 팝업
  │    ├─ 정렬 → 정렬 옵션 팝업
  │    ├─ 지도 → [현재위치 검색]
  │    ├─ 배너 탭 → 이벤트/랜딩
  │    └─ 숙소 카드 탭 → [숙소 상세]
  │
  ├─ [검색]
  │    ├─ 지역별 탭: 대분류 → 소분류 → [홈 (검색결과)]
  │    ├─ 역주변 탭: 도시 → 노선 → 역 → [홈 (검색결과)]
  │    ├─ 검색 입력 → [키워드 검색]
  │    │    ├─ 인기 키워드 → [홈 (필터)]
  │    │    ├─ 최근 검색어 → [검색결과] 또는 [숙소 상세]
  │    │    └─ 추천 결과 → [검색결과] 또는 [숙소 상세]
  │    └─ 지도 → [현재위치 검색]
  │
  ├─ [마일리지] ──→ 매장별 목록 → 마일리지 상세
  │
  ├─ [쿠폰] ──→ 쿠폰 목록 → 쿠폰 상세
  │
  └─ [더보기 (마이페이지)]
       ├─ 예약/이용 내역 → 예약 상세
       ├─ 비회원 예약확인 → 예약 상세
       ├─ 최근본/찜 → 숙소 상세
       ├─ 리뷰 → 리뷰 상세 → 리뷰 작성/수정
       ├─ 1:1 문의 → 문의 작성 / 문의 상세
       ├─ 이벤트 → 이벤트 상세 (쿠폰 다운로드)
       ├─ 공지사항 → 공지 상세
       ├─ FAQ → FAQ 상세
       ├─ 이용 가이드 → 가이드 상세
       ├─ 약관 → 약관 WebView
       ├─ 알림센터 → 알림 목록
       ├─ 설정 → 알림 토글, 앱잠금, 버전
       └─ 회원정보 → 프로필 수정 / 비밀번호 변경 / 로그아웃 / 탈퇴

[숙소 상세]
  ├─ 객실 탭 → [객실 상세]
  ├─ 리뷰 더보기 → 리뷰 목록
  ├─ 이벤트 → 이벤트 목록
  ├─ 지도 → 숙소 위치 (길찾기)
  ├─ 요금안내 → WebView
  ├─ 찜하기 (로그인 필요)
  └─ 공유 (카카오톡)

[객실 상세]
  └─ 예약 버튼 → [예약 Step1]

[예약 플로우]
  Step1: 시간 선택 + 예약자 정보(SMS 인증) + 교통수단
    → Step2: 쿠폰 + 마일리지 + 가격 + 결제수단 + 약관동의
      → PG 결제 (또는 현장결제/0원)
        → 예약 완료
```
