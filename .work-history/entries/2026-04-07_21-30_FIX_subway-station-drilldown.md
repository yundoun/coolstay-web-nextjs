# fix: 지하철 역 선택 3단계 드릴다운

## 문제
- 지하철 탭에서 호선 클릭 시 역 선택 없이 바로 날짜 단계로 넘어감
- API 데이터가 3레벨(그룹→호선→역) 구조인데 2레벨로만 파싱

## 수정
- `SearchModal.tsx`: subwayList → subwayGroups 3단계 파싱, 우측 패널 드릴다운 UI
- `SearchConditionBar.tsx`: 동일 3단계 구조 적용 (데스크톱 드롭다운)

## 동작
1. 좌측: 그룹 선택 (수도권, 대구, 부산)
2. 우측: 호선 목록 → 클릭 시 역 목록으로 전환 (← 뒤로가기 포함)
3. 역 클릭 시 선택 완료 → 날짜 단계로 진행
