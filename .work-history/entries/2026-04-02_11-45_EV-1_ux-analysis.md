## EV-1: 이벤트 영역 현행 UX 분석

**브랜치:** `feat/event-ui-improvement`

### 분석 방법
- dev 서버 실행 + Playwright로 홈/목록/상세 3개 화면 풀페이지 스크린샷 캡처
- 실제 API 데이터 기반 분석 (10개 이벤트: 진행중 1, 종료 9)

### 발견된 문제

#### 홈 EventSection
1. 종료 이벤트 노출 → 깨진 이미지 카드 나열, "죽은 섹션" 느낌
2. 카드 세로형(3:4) vs 배너형 이미지 → 텍스트 잘림
3. 중복 헤더 (Section 제목 + 내부 전체보기)

#### EventListPage
1. "종료" 중복 표시 (이미지 오버레이 + status dot)
2. "전체" 탭에 카운트 없음
3. 그리드 카드 4:3 aspect → 배너 이미지 잘림

#### EventDetailPage
1. 히어로 이미지 크롭 → 배너 텍스트 잘림
2. description 빈 경우 콘텐츠 영역 허전
3. -mt-4 겹침 너무 얕음
4. 데스크탑 CTA 페이지 하단에 위치 → 스크롤 없이 미노출
