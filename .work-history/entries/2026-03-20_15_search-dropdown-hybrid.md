# 헤더 검색 드롭다운 3단 하이브리드 + 인기검색어

- **날짜**: 2026-03-20 11:32
- **작업 유형**: enhancement
- **영향 범위**: src/components/layout/Header/CompactSearchBar.tsx

## 변경 내용

### 인기검색어 (빈 입력 포커스 시)
- 포커스만 하면 인기검색어 6개를 2열 그리드로 표시
- 1~3위는 primary 색상으로 강조

### 3단 자동완성 (타이핑 시)
- **검색어 추천**: 키워드 부분 매칭 (서 → 서면, 서울, 서귀포)
- **지역**: 📍(지역), 🚇(역) 아이콘 구분 (서울 전체, 서울역 등)
- **숙소**: 이미지 포함 숙소 미리보기
- 매칭 글자 하이라이트 (primary 색상)

### 기술 개선
- useMemo로 제안 연산 최적화
- 데이터를 상수로 분리 (KEYWORDS, REGIONS, ACCOMMODATIONS)
- 모바일 앱의 자동완성 UX를 웹 환경에 맞게 적용

## 참고
- 모바일 앱 캡쳐 기반으로 자동완성 UX 분석 후 구현
