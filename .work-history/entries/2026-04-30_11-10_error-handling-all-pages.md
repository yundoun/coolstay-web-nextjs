# 전체 페이지 서버 에러 처리 + 홈 스켈레톤 개선

## 변경 사항

### 서버 에러 처리 (7개 페이지)
- 검색, 숙소 상세, 기획전, 이벤트, 공지사항, FAQ, 매거진
- isError 상태 처리 추가 → "서버에 연결할 수 없습니다" + 다시 시도 버튼
- 기획전/이벤트 훅: JSON 에러 메시지 그대로 노출 → 사용자 친화적 메시지로 교체

### 홈 스켈레톤 개선
- 카테고리: flex 5개 → grid-cols-4 8개 (실제 UI와 일치)
- 배너: h-32 → aspect-[5/2] (실제 비율)
- 프로모 카드: 단순 사각형 → 아이콘+텍스트 구조
- 기획전 섹션 추가 (이전 누락)
- 이벤트: aspect-[2/1] → aspect-[2.5/1]
- RegionCardSkeleton 테두리/구분선 제거 (통일감)
