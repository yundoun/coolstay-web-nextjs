# 마이페이지 구현

## 작업 내용
- /mypage 페이지 구현 (모바일 앱 InformationFragment 기반)
- 프로필 섹션: 아바타 + 멤버십 뱃지 + 닉네임/이메일 + 설정 아이콘
- 쿠폰/마일리지 카드 (2열)
- 계정 메뉴: 예약/이용 내역, 최근/관심 숙소, 나의 후기, 나의 문의, 이용 약관
- 지원 메뉴: 공지사항(NEW 뱃지), 이벤트, 꿀팁 가이드, FAQ, 설정
- 로그아웃 버튼 + 앱 버전 표시

## 변경 파일
- src/domains/mypage/components/MyPage.tsx — 마이페이지 컴포넌트
- src/domains/mypage/components/index.ts — barrel export
